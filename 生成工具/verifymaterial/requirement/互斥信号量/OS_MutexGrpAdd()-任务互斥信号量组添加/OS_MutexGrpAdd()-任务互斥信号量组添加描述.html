需求描述

需求编号：HLR_OS_04_07

工作描述：HLR：uC/OS-Ⅲ应提供向任务的互斥量组添加成员的接口。

是否需求：是

验证方法：测试验证

补充信息：

1.函数名称

OS_MutexGrpAdd() ---向任务互斥量组添加成员的函数

2.函数框架

void OS_MutexGrpAdd (
    OS_TCB *p_tcb, /* 输入：指向任务控制块的指针 */
    OS_MUTEX *p_mutex /* 输入：指向互斥量控制块的指针 */
)

3.函数描述

该函数用于将一个互斥量添加到任务的互斥量组中。任务的互斥量组是一个链表，用于跟踪任务当前拥有的所有互斥量。

4.参数说明

    (1) p_tcb

        类型：OS_TCB*

        作用：指向任务控制块的指针，用于标识要添加互斥量的任务。

    (2) p_mutex

        类型：OS_MUTEX*

        作用：指向互斥量控制块的指针，用于标识要添加到任务互斥量组的互斥量。

5.函数算法逻辑

  1. 将互斥量的 MutexGrpNextPtr 指针设置为任务当前互斥量组的头部指针（p_tcb->MutexGrpHeadPtr）。
  2. 将任务的 MutexGrpHeadPtr 更新为当前互斥量的指针（p_mutex）。

7.返回值

    无

8.健康管理错误码

    无

备注：此函数是 uC/OS-III 的内部函数，应用程序不应直接调用它。。
