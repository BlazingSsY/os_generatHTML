需求描述

需求编号：HLR_OS_09_05

工作描述：HLR:uC/OS-Ⅲ应提供从延时列表中移除任务的接口

是否需求：是

验证方法：测试验证

补充信息：

1.函数名称

OS_TickListRemove() — 从延时列表中移除任务

2.函数框架

void OS_TickListRemove (
    OS_TCB *p_tcb /* 输入：待移除任务的任务控制块指针 */
)

3.函数描述

将指定任务从延时列表 OSTickList 中移除，并修正剩余节拍计数。

4.参数说明

(1) p_tcb

类型：OS_TCB*

作用：指向要移除的任务控制块。

5.函数算法逻辑

(1) 移除唯一节点

若目标任务是延时列表的唯一节点, 则将延时列表置为空，目标任务的剩余时间设置为0

(2) 移除有后续节点的头部节点

若目标任务是延时列表的头部节点且有后续节点：

• 设置目标任务的后续节点的前向指针为空，设置其为延时列表的头部节点，将其剩余时间加上目标任务的剩余时间

• 设置目标任务的后向指针为空，剩余时间为0

(3)移除非头部节点

若移除非头部节点：

• 将目标任务的前向节点的后向指针指向目标任务后向节点

• 若目标任务的后向节点不为空：

•  • 将目标任务的后向节点的前向指针指向前向节点，将后向节点的剩余事件加上目标任务的剩余时间

•将目标任务的前后向指针设置为空，剩余时间设置为0


6. 返回值

无

7.健康管理错误码

无
