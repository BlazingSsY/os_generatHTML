需求描述

需求编号：HLR_OS_02_07

工作描述：HLR：uC/OS-Ⅲ应提供消息队列的消息链表初始化接口，以便初始化消息队列的消息链表。

是否需求：是

验证方法：测试验证

补充信息：

1.函数名称

OS_MsgQInit()                       消息链表初始化                                

2.函数框架

void OS_MsgQInit (
    OS_MSG_Q *p_msg_q, /* 输入：指向消息链表的指针 */
    OS_MSG_QTY size /* 输入：消息链表中消息的最大数量 */
)

3.函数描述

OS_MsgQInit() 该函数用于初始化一个消息队列的消息链表。消息链表用于存储消息

4. 参数说明

(1) p_msg_q

类型：OS_MSG_Q*

作用：指向消息队列的消息链表的指针，用于管理消息链表。

(2) size

类型：OS_MSG_QTY

作用：指定消息链表中消息的最大数量。

5. 函数算法逻辑

（1）设置消息队列的最大消息数量

将消息链表的 NbrEntriesSize 成员设置为传入的 size 参数，表示消息队列可存储的最大消息数量。

（2）初始化消息队列的当前消息数量

将消息链表的 NbrEntries 成员初始化为 0，表示当前消息队列中没有消息。

（3）初始化消息队列的入队和出队指针

将消息链表的 InPtr 和 OutPtr 指针都设置为 NULL，表示消息队列为空。

6. 返回值

  无

7. 健康管理错误码

无
