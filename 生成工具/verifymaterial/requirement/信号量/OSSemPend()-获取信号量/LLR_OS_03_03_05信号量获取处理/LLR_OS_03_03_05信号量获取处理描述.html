需求描述

需求编号 ：LLR_OS_03_03_05

需求描述 ：

进入临界区后，检查信号量计数器 p_sem->Ctr：

  1. 若计数器大于 0 ，表示资源可用，立即获取资源（计数器减 1 ），然后退出临界区，将错误码 OS_ERR_NONE 存储到 p_err 中，并返回当前信号量计数。
  2. 若计数器等于 0 且等待选项为非阻塞模式，则将错误码 OS_ERR_PEND_WOULD_BLOCK 存储到 p_err 中，退出临界区并返回当前信号量计数。
  3. 若计数器等于 0 且等待选项为阻塞模式且调度器被锁定，则将错误码 OS_ERR_SCHED_LOCKED 存储到 p_err 中，退出临界区并返回 0 。
