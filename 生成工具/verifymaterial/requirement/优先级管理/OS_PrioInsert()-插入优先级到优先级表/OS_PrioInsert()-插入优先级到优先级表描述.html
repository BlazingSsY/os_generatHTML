需求描述

需求编号：HLR_OS_10_03

工作描述：HLR:uC/OS-Ⅲ应提供插入优先级到优先级表的接口

是否需求：是

验证方法：测试验证

补充信息：

1. 函数名称：OS_PrioInsert ()— 插入优先级

2. 函数框架

void OS_PrioInsert (OS_PRIO prio)

3. 函数描述

该函数用于在优先级位图表OSPrioTbl中插入指定优先级（即设置对应位为 1，表示该优先级下有任务就绪）。

4. 参数说明

(1 )prio:

  1. 类型：OS_PRIO
  2. 作用：需要插入的优先级（表示该优先级下有任务就绪）。

5. 函数算法逻辑

5.1 根据系统配置的最大优先级OS_CFG_PRIO_MAX与 CPU 数据宽度（CPU_CFG_DATA_SIZE * 8u）的关系来决定算法：

  1. 若OS_CFG_PRIO_MAX小于等于 2 倍 CPU 数据宽度：
  2. 若prio小于 CPU 数据宽度，在OSPrioTbl[0]中设置对应位
  3. 否则，在OSPrioTbl[1]中设置对应位

6. 返回值

类型：无返回值（void）

作用：修改优先级位图表OSPrioTbl，标记指定优先级下有任务就绪。

7. 健康管理错误码

无
