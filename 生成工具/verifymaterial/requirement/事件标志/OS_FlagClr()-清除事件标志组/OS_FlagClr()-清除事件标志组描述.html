需求描述

需求编号：HLR_OS_05_08

工作描述：HLR：uC/OS-Ⅲ应提供清除事件标志组的接口

是否需求：是

验证方法：测试验证

补充信息：

1. 函数名称

OS_FlagClr ()---事件标志组清除函数

    2. 函数框架

void OS_FlagClr (
    OS_FLAG_GRP *p_grp /* 输入：事件标志组控制块指针 */
);

    3. 函数描述

    此函数是内部函数，用于清除事件标志组的类型和标志位，并初始化其挂起列表。通常在删除事件标志组或需要重置事件标志组状态时调用。

    4. 参数说明

        (1) p_grp

        类型：OS_FLAG_GRP*

        作用：指向要清除的事件标志组控制块

    5. 函数逻辑

  1. 如果OS_OBJ_TYPE_REQ功能启用 ，将事件标志组的 Type 成员设置为 OS_OBJ_TYPE_NONE，表示该对象类型被重置。
  2. 将事件标志组的 Flags 成员设置为 0，清除所有事件标志位。
  3. 调用 OS_PendListInit() 函数初始化挂起列表

   6. 返回值

    无

   6. 健康管理错误码

    无

    7. 注意事项

    此函数是 uC/OS-III 的内部函数，不能被用户直接调用
