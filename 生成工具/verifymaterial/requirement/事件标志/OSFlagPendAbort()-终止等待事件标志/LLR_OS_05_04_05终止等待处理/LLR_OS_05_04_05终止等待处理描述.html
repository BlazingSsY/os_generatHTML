需求描述

需求编号：LLR_OS_05_04_05

需求描述：

  1. 进入临界区，获取事件标志组的挂起列表 p_pend_list。
  2. 检查挂起列表的头部指针是否为空。若为空，说明没有任务在等待该事件标志组，退出临界区，将错误码 OS_ERR_PEND_ABORT_NONE 存储到 p_err 中，并返回 0 。
  3.  初始化任务计数 nbr_tasks 为 0 和时间戳变量 ts 为 0 。
  4. 循环遍历挂起列表，只要挂起列表的头部指针不为空，就获取当前挂起列表头部的 TCB 指针 p_tcb，调用 OS_PendAbort() 函数来终止该任务的等待操作，设置任务的 pend 状态为 OS_STATUS_PEND_ABORT，同时任务计数 nbr_tasks 增加 1 。如果 opt 不包含中止所有任务选项，则在中止一个任务后跳出循环。
  5. 退出临界区
