需求描述

需求编号：HLR_OS_07_04

工作描述：HLR：uC/OS-Ⅲ应提供内存池管理器初始化的接口。

是否需求：是

验证方法：测试验证

补充信息：

1.函数名称

OS_MemInit()---内存池管理器初始化

2.函数框架

void OS_MemInit (
    OS_ERR *p_err /* 输出：错误码指针 */
)

3.函数描述

该函数用于初始化内存池管理器。目前只是一个简单的占位函数，将错误码设置为无错误

4.参数说明

    (1) ) p_err

        类型：OS_ERR*

        作用：用于返回函数执行的错误代码，指示函数调用是否成功以及失败的原因。

        错误码：

OS_ERR_NONE：函数执行成功


5.函数算法逻辑

（1）设置错误码

  1. 将错误码 OS_ERR_NONE存储到 p_err中，表示内存池管理器初始化成功。

6.返回值

无

7.健康管理错误码

无

备注：此函数是内部函数，用户不可直接调用
