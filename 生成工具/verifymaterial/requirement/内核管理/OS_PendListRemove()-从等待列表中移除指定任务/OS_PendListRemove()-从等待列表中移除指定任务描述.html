需求描述

需求编号：HLR_OS_11_18

工作描述：HLR：uC/OS-Ⅲ应提供从等待列表中移除指定任务的接口。

是否需求：是

验证方法：测试验证

补充信息：

1.函数名称

OS_PendListRemove() – 从等待链表中移除指定任务

2.函数框架

void OS_PendListRemove (
    OS_TCB *p_tcb /* 输入：待移除的任务控制块 */
)

3.函数描述

该函数用于根据任务控制块从其所属等待列表中移除该任务节点，移除后，将更新等待列表头尾指针与相邻任务节点指针，确保列表结构完整。同时，清空任务控制块中与等待列表相关的指针。

4.参数说明

（1）p_tcb

  1. 类型：OS_TCB*
  2. 作用：指向需要从其等待链表中移除的任务控制块。
  3.  注意事项：该任务控制块的 PendObjPtr 成员需指向一个有效的挂起对象，否则函数不执行任何操作。

5.算法逻辑

  1. 检查任务是否有等待对象
  2.  •若任务的等待对象为空，表示不在任何挂起链表中，函数直接返回。
  3. 从等待对象的等待链表中移除指定任务
  4.  •获取任务等待对象的等待列表
  5.  •若等待列表只有一个节点，则将等待列表的头尾指针置空。
  6.  •若指定任务是等待列表的表头任务，则将指定任务的下一个任务设置为新的表头任务，将新表头任务的前向指针置空。
  7.  •若指定任务是等待列表的表尾任务，则将指定任务的前一个任务设置为新的表尾任务，将新表尾任务的后向指针置空。
  8.  •若指定任务是等待列表的中间节点，将指定任务在等待列表中的前一个任务的后向指针指向指定任务的后一个任务，将指定任务在等待列表中的后一个任务的前向指针指向指定任务的前一个任务。
  9.  •清除任务控制块的前向指针，后向指针和等待对象指针。

6.返回值

无。

7.健康管理错误码

无。

备注：此函数为 uC/OS-III 的内部函数，您的应用程序不得调用它。
