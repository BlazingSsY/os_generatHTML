需求描述

需求编号：HLR_OS_11_07

工作描述：HLR：uC/OS-III 应提供配置时间片轮询调度机制的接口，该接口应能全局启用/禁用轮询机制并设置默认时间片长度。

是否需求：是

验证方法：测试验证


1 函数名称

OSSchedRoundRobinCfg() - 配置时间片轮询机制


2 函数框架

void OSSchedRoundRobinCfg(
    CPU_BOOLEAN en,           /* 输入: 启用/禁用时间片轮询机制 */
    OS_TICK dflt_time_quanta, /* 输入: 默认时间片长度（时钟节拍数） */
    OS_ERR *p_err             /* 输出: 错误码指针 */
)


3 函数描述

该函数用于全局配置时间片轮询调度机制，可在运行时动态设置轮询机制的启用状态和默认时间片长度，配置后立即生效，影响系统中所有参与轮询调度的任务。


4 参数说明

（1）en

类型：CPU_BOOLEAN

作用：用于启用/禁用时间片轮询机制


（2）dflt_time_quanta

类型：OS_TICK

作用：用于设置默认时间片长度（单位：系统时钟节拍）


（3）p_err

类型：OS_ERR*

作用：用于返回函数执行的错误代码，指示函数调用是否成功以及失败的原因

错误码：目前只有OS_ERR_NONE一种错误码


5 算法逻辑

以下操作需在临界区中进行，即在操作前进入临界区，操作完成后退出临界区

1）启用/禁用时间片轮询机制

  1. 若输入的en等于0，则设置OSSchedRoundRobinEn = OS_FALSE，标志全局禁用时间片轮询机制；否则，设置设置 OSSchedRoundRobinEn = OS_TRUE，标志全局启用时间片轮询机制

2）设置默认时间片长度

  1. 若输入dflt_time_quanta > 0，则设置 OSSchedRoundRobinDfltTimeQuanta = dflt_time_quanta；否则，设置OSSchedRoundRobinDfltTimeQuanta 为心跳频率的十分之（OSCfg_TickRate_Hz / 10）
  2. 设置 *p_err = OS_ERR_NONE


6 返回值

无


7 健康管理错误码

无
