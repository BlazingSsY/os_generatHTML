需求描述

需求编号：HLR_OS_11_12

工作描述： HLR：uC/OS-III 应提供初始化并创建空闲任务的接口

是否需求： 是

验证方法： 测试验证

补充信息：

1.函数名称

OS_IdleTaskInit() - 空闲任务初始化

2.函数框架

void OS_IdleTaskInit(
OS_ERR *p_err /* 输出：错误码指针 */
)

3.函数描述

该函数用于初始化并创建系统的空闲任务(Idle Task)。

4.参数说明

参数名称：p_err

参数类型：OS_ERR*

作用：输出参数，用于传递任务创建是否成功的错误代码

错误代码: 与OSTaskCreate函数的错误相同

5.算法逻辑

1)若空闲任务功能启用 （OS_CFG_TASK_IDLE_EN > 0）, 使用系统配置的空闲任务参数，调用 OSTaskCreate()函数，初始化空闲任务控制块并创建空闲任务


6.返回值

无

7.健康管理错误码

  1. 参数：p_err
  2. 类型：OS_ERR*
  3. 返回值：OS_ERR_NONE

备注：

此功能为 uC/OS-III 内部使用，应用程序不能直接调用。
