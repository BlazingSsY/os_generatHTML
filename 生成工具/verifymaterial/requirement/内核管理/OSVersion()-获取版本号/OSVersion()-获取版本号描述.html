需求描述

需求编号： HLR_OS_11_10

工作描述： HLR：uC/OS-III 应提供获取内核版本号的接口。

是否需求： 是

验证方法： 测试验证

1. 函数名称

OSVersion() – 获取内核版本号

2. 函数框架

CPU_INT16U OSVersion (
    OS_ERR *p_err  /* 输出：错误码指针 */
)

3. 函数描述

该函数用于返回 uC/OS-III 内核的版本号。返回值为系统版本号的宏定义OS_VERSION。

4. 参数说明

（1）p_err

  1. 类型： OS_ERR*
  2. 作用： 用于返回函数调用结果的状态码。

5. 算法逻辑

1）设置错误码并返回版本号

  1. 设置 *p_err = OS_ERR_NONE，表示操作成功。
  2. 返回常量 OS_VERSION，表示当前操作系统的版本号

6. 返回值

类型：CPU_INT16U

含义：返回当前 uC/OS-III 的版本号乘以 10000 的值。


7. 健康管理错误码

  1. 无

备注：

 该函数为只读查询型函数，不会更改系统状态，可在任何时刻调用，适用于版本识别、兼容性验证等用途。
