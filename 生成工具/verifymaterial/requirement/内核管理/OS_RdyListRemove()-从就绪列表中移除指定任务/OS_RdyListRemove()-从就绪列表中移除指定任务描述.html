需求描述

需求编号：HLR_OS_11_25

工作描述：HLR：uC/OS-Ⅲ应提供从就绪列表中移除指定任务的接口

是否需求： 是

验证方法： 测试验证

补充信息

1. 函数名称

OS_RdyListRemove() - 从就绪列表中移除指定任务

2. 函数框架

void OS_RdyListRemove (
    OS_TCB *p_tcb /* 输入：指向待移除的任务控制块的指针 */
)

3. 函数描述

该函数用于从就绪列表中移除指定的任务，根据传入的任务控制块指针，在就绪列表组中找到对应优先级的就绪列表，然后将任务从该就绪列表中删除。

4. 参数说明

（1）p_tcb

  1. 类型：OS_TCB*
  2. 作用：指向待从就绪列表中移除的任务控制块
  3. 注意事项：必须为非空指针，且指向已存在于 OS_RDY_LIST 的 TCB 节点。

5. 算法逻辑

  1. 根据目标任务优先级从就绪列表组中获取该任务所在的就绪列表
  2. 从该优先级就绪列表中移除目标任务，需要对以下三种情况做出对应处理：
  3.  •该就绪列表中只有目标任务，则从优先级表移除该优先级，清空该优先级就绪列表
  4.  •该就绪列表有多个任务且目标任务为列表头部节点，则设置目标任务的下一个任务为头部任务，将新的头部任务的前向指针置空
  5.  •该就绪列表有多个任务且目标任务不是头部任务节点，则设置目标任务前一个任务的后向指针指向目任务的后一个任务，同时检查目标任务是否是就绪列表尾部，若是，则设置就绪列表的列尾指针为空，否则设置目标任务的后一个任务指向目标任务的前一个任务

最后，清空目标任务 的 PrevPtr 和 NextPtr

6. 返回值

无

7. 健康管理错误码

无

备注

此函数是 uC/OS-III 的内部函数，您的应用程序不得直接调用该函数。通常由任务调度或任务挂起、删除等内部机制调用。
