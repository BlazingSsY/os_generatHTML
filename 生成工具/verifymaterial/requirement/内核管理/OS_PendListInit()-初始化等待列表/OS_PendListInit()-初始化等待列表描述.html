需求描述

需求编号： HLR_OS_11_16

工作描述： HLR：uC/OS-III 应提供初始化等待列表的接口。

是否需求： 是

验证方法： 测试验证

1. 函数名称

OS_PendListInit() – 初始化等待列表

2. 函数框架

void OS_PendListInit (
    OS_PEND_LIST *p_pend_list /* 输入：指向等待列表结构体的指针 */
)

3. 函数描述

该函数用于初始化等待列表，将等待列表的头指针和尾指针清空。

4. 参数说明

(1)参数名：p_pend_list

  1. 类型：OS_PEND_LIST*
  2. 作用：指向需要初始化的等待列表

5. 算法逻辑

  1. 将等待列表的头指针置空。
  2. 将等待列表的尾指针置空。

6. 返回值

无。

7. 健康管理错误码

无。

备注：

 此函数是 uC/OS-III 内部使用函数，应用程序不得直接调用。
