需求描述

 需求编号：HLR_OS_11_23

 工作描述：HLR：uC/OS-III应提供将任务插入就绪列表尾部的接口。

 是否需求：是

 验证方法：测试验证

 补充信息：

 1.函数名称

 OS_RdyListInsertTail() - 将任务插入就绪列表尾部

2.函数框架

void OS_RdyListInsertTail (
    OS_TCB *p_tcb /* 输入：待插入的任务控制块指针 */
)

3.函数描述

 该函数用于将任务插入其对应优先级的就绪列表的尾部，插入尾部使该任务成为当前优先级中最末尾等待执行的任务。

4.参数说明

 （1）p_tcb

  1. 类型：OS_TCB*
  2. 作用：指向需要插入的任务控制块。

5.算法逻辑

 1）根据目标任务的优先级，从就绪列表组中获取对应的就绪列表，并判断就绪列表是否为空:

  1. 若就绪列表为空：
  2.  • 将就绪列表的头尾指针都设置为指向目标任务，将目标任务的前向指针和后向指针设置为空
  3. 若就绪列表为非空，
  4.  •将目标任务的后向指针设置为空，设置其前向指针指向就绪列表的尾部任务
  5.  •将尾部任务的后向指针指向目标任务。
  6.  •设置就绪列表的尾部指针指向目标任务

6.返回值

 无。

7.健康管理错误码

 无。

备注：此函数为uC/OS-III内部使用，应用程序不得直接调用。
