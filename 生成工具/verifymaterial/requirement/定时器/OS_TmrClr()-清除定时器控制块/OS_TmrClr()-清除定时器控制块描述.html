需求描述

需求编号：HLR_OS_06_08

工作描述：HLR：uC/OS-Ⅲ应提供清除定时器控制块的功能。

是否需求：是

验证方法：测试验证

补充信息：

1.函数名称

OS_TmrClr() --- 清除定时器控制块

2.函数框架

void OS_TmrClr (
    OS_TMR *p_tmr /* 输入：指向定时器控制块的指针 */
)

3.函数描述

该函数用于清空定时器控制块（TCB）的所有字段，将其重置为初始未使用状态（OS_TMR_STATE_UNUSED）。通常在删除定时器（OSTmrDel）时调用，确保资源可重复利用或释放。

4.参数说明

（1）p_tmr

类型：OS_TMR*

作用：指向要清除的定时器控制块。

注意事项：必须指向有效内存，不可为 NULL；调用前需确保定时器已停止且未被其他任务引用。

5.算法逻辑

1）清除定时器控制块

将定时器的状态（State）设置为 OS_TMR_STATE_UNUSED，表示定时器处于未使用状态。若系统要求对象类型（OS_OBJ_TYPE_REQ > 0u），将定时器的类型（Type）设置为 OS_OBJ_TYPE_NONE。

  1. 将定时器的启动延迟时间（Dly）、剩余时间（Remain）、周期时间（Period）均设置为 0。
  2. 将定时器的选项（Opt）设置为 0。
  3. 将定时器的回调函数指针（CallbackPtr）设置为空（NULL）。
  4. 将定时器的回调函数参数（CallbackPtrArg）设置为空（NULL）。
  5. 将定时器的下一个指针（NextPtr）和前一个指针（PrevPtr）均设置为空（NULL）。

6.返回值

无。

7.健康管理错误码

无。

备注：此函数是uC/OS-III的内部函数，您的应用程序不得调用它。
